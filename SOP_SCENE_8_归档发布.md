# 场景8：归档发布

## 触发条件
用户确认发布或要求归档

## 执行步骤

**Step 1: 发布到公众号（如果选择了）**

如果用户选择了包含「发布到公众号」的选项，先执行场景7.5的流程。

**Step 2: 移动到知识库**
```
将云文档移动到知识库：
- 目标目录：/已发布文章/[自动分类]/
```

**Step 3: 更新记录**
```json
{
  "阶段": "已发布",
  "知识库链接": {"link": "URL", "text": "查看知识库"},
  "发布时间": "毫秒时间戳",
  "发布平台": ["公众号", "小红书"],
  "草稿ID": "微信草稿media_id（如果有）"
}
```

**Step 4: 更新素材库**
```json
{"状态": "已发布"}
```

**Step 5: 反馈用户**
```
🎉 发布完成！

📁 已归档到知识库：[分类目录]
🔗 知识库链接：[知识库链接]

📱 公众号草稿：
   - 草稿链接：[预览链接]
   - 请在公众号后台检查并发布

本次内容生产完成 ✅
```
