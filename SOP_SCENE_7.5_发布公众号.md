## 关键命令速查

```bash
# 1. Markdown 转 HTML（获取 prompt，默认 AI 模式）
md2wechat convert article.md --json

# 2. 🎨 AI 生成封面图
cd ~/.openclaw/workspace-content/skills/ai-cover-generator
python generate_cover.py --title "文章标题" --output ./cover.jpg --json

# 3. 上传图片
md2wechat upload_image ./cover.jpg --json
md2wechat download_and_upload https://example.com/image.jpg --json

# 4. 创建草稿
md2wechat create_draft draft.json --json

# 5. 查看可用主题
md2wechat themes --json
```

## 封面图生成说明

**自动风格匹配**：根据文章标题关键词自动选择风格

| 关键词 | 风格 |
|--------|------|
| AI、人工智能 | 科技感，蓝色调，神经网络图案 |
| 教程、如何 | 简约现代风格，步骤图示 |
| 案例、复盘 | 商务专业风格，数据图表 |
| 观点、思考 | 抽象艺术风格，对比色块 |
| 方法、技巧 | 实用工具风格，技巧提示 |

**封面图尺寸**：默认 900x383 (2.35:1)，公众号官方推荐封面大图尺寸

## 注意事项

1. **封面图片**：创建草稿必须有封面图片
   - 优先使用用户提供的封面
   - 若无封面，AI 自动生成
   - 需要配置 SILICONFLOW_API_KEY
2. **图片上传**：微信图片需先上传到素材库获取 media_id
3. **IP白名单**：服务器 IP 需要在微信公众号后台添加到白名单
4. **审核时间**：草稿创建后可在公众号后台预览和发布

## 异常处理

| 异常情况 | 处理方式 |
|---------|---------|
| 未配置微信凭证 | 提示用户配置 WECHAT_APPID/SECRET |
| 未配置硅基流动 API Key | 提示用户配置 SILICONFLOW_API_KEY 或手动提供封面 |
| IP 不在白名单 | 告知需要添加的 IP 地址 |
| 封面图生成失败 | 重试 3 次，失败则提示用户手动上传 |
| 图片上传失败 | 跳过该图片，使用占位符，后续手动替换 |
