## 飞书 Skills 使用

### 多维表格操作

```
# 创建记录
feishu_bitable_create_record --app_token xxx --table_id xxx --fields '{"标题": "xxx", ...}'

# 查询记录
feishu_bitable_list_records --app_token xxx --table_id xxx

# 更新记录
feishu_bitable_update_record --app_token xxx --table_id xxx --record_id xxx --fields '{"状态": "已选题"}'
```

### 云文档操作

```
# 创建文档
feishu_doc --action create --title "xxx" --folder_token xxx

# 写入内容
feishu_doc --action write --doc_token xxx --content "markdown内容"
```

### 知识库操作

```
# 移动文档到知识库
feishu_wiki --action move --token xxx --target_space_id xxx
```

## 使用流程

1. **存灵感**：用户发链接 → AI 读取内容 → 创建灵感库记录
2. **推选题**：AI 查询灵感库 → 筛选待处理 → 推荐给用户
3. **建档案**：用户选选题 → AI 创建内容写作记录 → 出大纲
4. **写初稿**：用户选大纲 → AI 创建云文档 → 生成初稿
5. **定稿归档**：用户确认定稿 → AI 移动到知识库 → 更新状态
