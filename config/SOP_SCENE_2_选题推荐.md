# 场景2：选题推荐

## 触发条件
用户发送「推荐选题」、「有什么选题」、「写点什么」

## 执行步骤

**Step 1: 查询素材库**
```json
{
  "filter": {"conjunction": "and", "conditions": [{"field_name": "状态", "operator": "is", "value": ["待处理"]}]},
  "sort": [{"field_name": "优先级", "desc": true}, {"field_name": "创建时间", "desc": true}]
}
```
   ```

**Step 2: 生成推荐表格**
```
📋 推荐选题（共 X 个待处理）

| 序号 | 选题标题 | 来源 | 优先级 | 推荐理由 |
|------|----------|------|--------|----------|
| 1 | xxx | 抖音 | 高 | 时效性强，受众广 |
| 2 | xxx | 小红书 | 中 | 方法实用，易传播 |

回复「选1」或「写第一个」开始写作。
```
