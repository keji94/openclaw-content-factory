## 可配置的定时任务

### 每日任务

```
时间：每天 09:00

任务：
1. 检查「灵感库」中待处理数量
2. 如果超过 10 条，提醒用户筛选
3. 检查「内容写作」中未完成的记录
```

### 每周任务

```
时间：每周一 10:00

任务：
1. 统计上周灵感入库数量
2. 统计上周完成文章数量
3. 推荐本周重点选题
4. 同步已发布文章到 Obsidian（如果配置了）
```

---

## 当前状态

定时任务未启用。如需启用，请在 OpenClaw 配置中设置。
